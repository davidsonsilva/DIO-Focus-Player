Política de Privacidade

Última atualização: 16 de julho de 2026. Esta página descreve como o DIO Focus Player trata dados quando a extensão é usada para adaptar localmente o layout das aulas da DIO.

Resumo rápido

A extensão não possui backend, não coleta telemetria, não exibe anúncios e não compartilha dados com terceiros. O processamento acontece no navegador do usuário e o armazenamento é local.

Conformidade

A extensão foi desenhada para respeitar o princípio de uso limitado: somente o necessário para cumprir o propósito único de adaptar o layout da aula é processado no dispositivo.

Dados processados localmente

  • estado do modo foco;
  • preferência de ativação manual ou automática;
  • limite de largura configurado pelo usuário;
  • preferência de exibição da lista de aulas;
  • estado técnico local da extensão na página aberta.

O que não é coletado

  • cookies;
  • tokens de autenticação;
  • conteúdo integral das aulas;
  • histórico de navegação;
  • telemetria, analytics ou anúncios.

Armazenamento e rede

As preferências são salvas apenas no armazenamento local do navegador. A extensão não executa chamadas de rede nem envia dados para servidores externos.

Permissões

  • storage: salvar preferências localmente;
  • activeTab: identificar a aba ativa quando o popup é aberto;
  • https://web.dio.me/*: aplicar o modo foco apenas nas páginas da DIO.

Controle do usuário

O usuário pode desativar o modo foco pelo popup, desinstalar a extensão a qualquer momento e recarregar o item no Chrome para limpar as preferências locais.

Contato

O canal de suporte e as informações de publicação ficam centralizados no repositório público do projeto, na página principal e nas issues do GitHub.

Declaração para a Chrome Web Store

O DIO Focus Player limita o uso das informações necessárias ao seu propósito único de adaptar localmente o layout das aulas. O projeto não usa os dados para publicidade, rastreamento, revenda ou qualquer finalidade diferente da funcionalidade descrita.

DIO Focus Player é um projeto independente e não oficial. Esta página existe para fornecer a política pública exigida na publicação e pode ser vinculada no campo de privacidade da Chrome Web Store.
